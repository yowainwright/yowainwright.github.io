# Fix

Quick fix the current error or issue without a full review. Focus on the immediate problem only.

## Instructions

1. Identify the specific error or issue mentioned
2. Find the root cause (don't just fix symptoms)
3. Apply the minimal fix required
4. Verify the fix doesn't break related code

## Rules

- **Minimal changes only** - Don't refactor surrounding code
- **Stay focused** - Fix only what's asked, nothing else
- **Explain briefly** - One sentence on what was wrong and how you fixed it
- **No side quests** - Don't improve code style, add types, or clean up unless directly related to the bug

## Output Format

```
**Problem**: [One sentence describing the issue]
**Fix**: [One sentence describing the solution]
**Changed**: [File:line]
```

Then make the edit.
