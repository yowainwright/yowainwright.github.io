# Add Rule

Add a coding style rule to the JavaScript/React Quick Reference in CLAUDE.md.

## Usage
/add-rule <rule text>

## Instructions

1. Read ~/.claude/CLAUDE.md
2. Find the `### JavaScript/React` section
3. Append the new rule as a bullet point: `- <rule text>`
4. Save the file
5. Confirm briefly: "Added: <rule>"

Keep the rule concise. Match existing bullet style.
