## Setting up

Install this starter (assuming Gatsby is installed) by running from your CLI:
`gatsby new gatsby-blog https://github.com/gatsbyjs/gatsby-starter-blog`

## Development

### Building
```shell
gatsby build
```

### Developing with reload

```shell
gatsby develop
```

### Deployment with necessary files

```shell
npm run deploy
```
